# Model of Consequence-Based Decision-Making

Implementation in Matlab of the mathematical model published in Cecchini G., DePass M., Baspinar E., Andujar M., Ramawat S., Pani P., Ferraina S., Destexhe A., Moreno-Bote R., Cos I. A Theoretical Formalization of Consequence-Based Decision-Making. bioRxiv 2023.02.14.528595

To run the model, download all the files contained in this folder, and save them in your computer. Open matlab, set current folder to the one where you save the files, and run the main script called script_main.m. Follow the instructions displayed in the command window.
